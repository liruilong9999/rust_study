# rust_study

## 笔记简介

这是一份按阶段整理的 Rust（Rust 编程语言）学习笔记。内容以示例代码配合中文说明文档的形式展开，整体更像持续补充的学习笔记，只是由 AI（人工智能）辅助总结和整理，方便按主题阅读、复盘和回顾。

## 学习导航

建议按照下面顺序学习：

1. [a00_install 安装与工具链](code/a00_install/a00_install_学习笔记.md)
2. [a01_helloworld 第一阶段：程序入口与输出](code/a01_helloworld/a01_helloworld_学习笔记.md)
3. [a02_guessing_name 第二阶段：输入、循环、匹配与类型转换](code/a02_guessing_name/a02_guessing_name_学习笔记.md)

## 目录说明

当前 `code` 目录按学习阶段整理：

- [a00_install](code/a00_install/a00_install_学习笔记.md)：安装 Rust、认识 `rustup`、掌握 `cargo` 常用命令与包管理基础
- [a01_helloworld](code/a01_helloworld/a01_helloworld_学习笔记.md)：学习 `fn main`、`println!`、字符串字面量、基础运行方式
- [a02_guessing_name](code/a02_guessing_name/a02_guessing_name_学习笔记.md)：学习 `use`、`let mut`、随机数、输入读取、`loop`、`match`、`Result`

## 学习方法建议

每学完一个阶段，建议按下面顺序复盘：

1. 先运行示例代码，确认自己知道代码做了什么。
2. 再阅读同目录下的阶段笔记文件，整理本阶段的重点内容。
3. 如果你新增了代码示例，也要同步补文档和根目录导航。
