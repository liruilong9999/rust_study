# rust_study

## 笔记简介

这是一份按阶段整理的 Rust（Rust 编程语言）学习笔记。每个阶段都会配套示例代码和中文说明文档，方便按章节学习、动手运行和回顾重点。

## 学习路径总览

当前建议按下面顺序学习：

1. [a00_install 安装与工具链](code/a00_install/a00_install_学习笔记.md)
2. [a01_helloworld 第一阶段：程序入口与输出](code/a01_helloworld/a01_helloworld_学习笔记.md)
3. [a02_guessing_name 第二阶段：输入、循环、匹配与类型转换](code/a02_guessing_name/a02_guessing_name_学习笔记.md)
4. [a03_concepts 第三阶段：常用编程概念](code/a03_concepts/a03_concepts_学习笔记.md)

## 阶段入口与核心语法摘要

- [a00_install](code/a00_install/a00_install_学习笔记.md)：环境安装、`rustup`、`cargo` 常用命令、包管理基础
- [a01_helloworld](code/a01_helloworld/a01_helloworld_学习笔记.md)：`fn main`、`println!`、字符串字面量、基础运行方式
- [a02_guessing_name](code/a02_guessing_name/a02_guessing_name_学习笔记.md)：`use`、`let mut`、输入读取、`loop`、`match`、`Result`
- [a03_concepts](code/a03_concepts/a03_concepts_学习笔记.md)：变量与可变性、数据类型、函数、注释、`if`、`loop`、`while`、`for`

## 目录说明

当前 `code` 目录按学习阶段整理：

- `code/a00_install`：环境安装、工具链、包管理、Cargo 命令
- `code/a01_helloworld`：第 1 阶段，基础程序入口与输出
- `code/a02_guessing_name`：第 2 阶段，输入、循环、匹配、类型转换
- `code/a03_concepts`：第 3 阶段，变量、数据类型、函数、注释、控制流

## 建议学习顺序

每学完一个阶段，建议按下面顺序复盘：

1. 先运行示例代码，确认自己知道代码做了什么。
2. 再阅读同目录下的阶段笔记文件，整理本阶段重点。
3. 最后回到根目录 `README.md`，把这一阶段放进整体学习路径里理解。
