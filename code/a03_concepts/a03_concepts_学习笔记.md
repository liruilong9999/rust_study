# a03_concepts 学习笔记

## 1. 阶段目标

这一阶段开始系统接触 Rust（Rust 编程语言）里的常用编程概念，重点不再只是“把程序跑起来”，而是理解写程序时最常见的语法结构和思维方式。

本章内容覆盖：

- 变量与可变性
- 数据类型
- 函数
- 注释
- 控制流

## 2. 本阶段核心语法

### 2.1 变量与可变性

Rust 默认变量是不可变的：

```rust
let x = 5;
```

如果想修改变量值，需要加 `mut`：

```rust
let mut x = 5;
x = 6;
```

这一节还要分清两个概念：

- `mut` 是“同一个变量允许修改值”
- 遮蔽（Shadowing，遮蔽）是“重新定义一个同名变量”

例如：

```rust
let spaces = "   ";
let spaces = spaces.len();
```

这里不是把字符串原地改成数字，而是创建了一个新的同名变量，所以类型可以变化。

### 2.2 数据类型

Rust 常见数据类型分两类：

- 标量类型：整数、浮点数、布尔值、字符
- 复合类型：元组、数组

#### 整数类型

常见整数类型：

- 有符号：`i8`、`i16`、`i32`、`i64`、`i128`、`isize`
- 无符号：`u8`、`u16`、`u32`、`u64`、`u128`、`usize`

如果 Rust 无法自动推断类型，可以显式标注：

```rust
let guess: u32 = "42".parse().expect("不是有效的数字");
```

#### 浮点数

Rust 的浮点数默认是 `f64`，也可以手动写成 `f32`：

```rust
let x = 2.0;
let y: f32 = 3.0;
```

#### 布尔与字符

- 布尔类型是 `bool`，只有 `true` 和 `false`
- 字符类型是 `char`，使用单引号，例如 `'z'`

#### 元组

元组（Tuple，元组）可以把多个不同类型的值放在一起：

```rust
let tup = (500, 6.4, 1);
let (x, y, z) = tup;
```

也可以通过下标访问：

```rust
let first = tup.0;
```

#### 数组

数组适合存放固定长度、同一类型的数据：

```rust
let a = [1, 2, 3, 4, 5];
```

需要注意：

- 数组长度固定
- 数组元素类型必须一致
- 下标越界会触发 panic（程序异常终止）

### 2.3 函数

函数使用 `fn` 定义：

```rust
fn main() {
}
```

带参数的函数要写清楚参数类型：

```rust
fn another_function(x: i32) {
    println!("参数 x 的值是：{x}");
}
```

#### 语句与表达式

这一节是本章最关键的理解点之一：

- 语句（Statement，语句）执行动作，一般不返回值
- 表达式（Expression，表达式）会计算出一个值

例如块表达式：

```rust
let y = {
    let x = 3;
    x + 1
};
```

最后一行没有分号，所以它是返回值。如果写成 `x + 1;`，值就被丢弃了。

#### 返回值

Rust 常用“最后一个表达式”作为返回值：

```rust
fn five() -> i32 {
    5
}
```

如果末尾加了分号，就不再返回该值：

```rust
fn plus_one(x: i32) -> i32 {
    x + 1
}
```

### 2.4 注释

本章涉及三类注释：

- 行注释：`//`
- 块注释：`/* ... */`
- 文档注释：`///` 和 `//!`

说明：

- `///` 常用于说明后面的函数、结构体或模块成员
- `//!` 常用于说明整个模块或整个文件
- Rust 支持嵌套块注释

### 2.5 控制流

控制流主要包含 `if` 和循环。

#### `if` 表达式

Rust 的 `if` 条件必须是 `bool`：

```rust
if number != 0 {
    println!("number 不是 0");
}
```

Rust 不会像某些语言那样自动把 `0`、`1` 之类的整数转换成布尔值。

`if` 还可以直接放到 `let` 右边：

```rust
let number = if condition { 5 } else { 6 };
```

但两边分支的类型必须一致。

#### 循环

Rust 常见循环有三种：

- `loop`：无限循环，通常搭配 `break`
- `while`：条件循环
- `for`：遍历集合或区间

`loop` 有一个很重要的特点：它本身是表达式，可以返回值。

```rust
let result = loop {
    counter += 1;
    if counter == 10 {
        break counter * 2;
    }
};
```

这里的重点是：

- 返回值来自 `break` 后面的表达式
- 分号只是结束 `break` 这条语句
- 真正接收这个值的是外层的 `loop` 表达式

这也是为什么 `let result = loop { ... };` 可以成立。

#### 循环标签

嵌套循环里可以给循环加标签（Label，标签）：

```rust
'counting_up: loop {
    loop {
        break 'counting_up;
    }
}
```

含义是直接退出名为 `'counting_up` 的外层循环，而不是只退出当前这一层。

#### `for` 与 `rev()`

当前启用的示例代码是：

```rust
for number in (1..4).rev() {
    println!("{number}！");
}
```

说明：

- `1..4` 是左闭右开区间，包含 `1`、`2`、`3`
- `rev()` 是 reverse（反转）迭代顺序
- 所以输出顺序是 `3`、`2`、`1`

## 3. 本章易错点整理

### 3.1 `mut` 和遮蔽不是一回事

- `mut` 修改的是同一个变量绑定
- 遮蔽会创建新的同名变量绑定
- 遮蔽允许类型变化，`mut` 不允许直接把变量改成另一种类型

### 3.2 语句和表达式要分清

- `let y = 6;` 是语句
- `6`、`x + 1`、`if ... else ...`、`loop { ... }` 都可以是表达式
- 函数返回值和块返回值都依赖“最后一个没有分号的表达式”

### 3.3 `if` 分支类型必须一致

下面这种写法会报错：

```rust
let number = if condition { 5 } else { "six" };
```

因为一个分支是整数，另一个分支是字符串，类型不兼容。

### 3.4 数组访问要注意越界

```rust
let element = a[index];
```

如果 `index` 超出数组范围，程序会 panic，所以遍历数组时通常更推荐 `for`。
